# Anjali Kannan Portfolio Website

## Files included
- `index.html`
- `styles.css`
- `assets/` folder with your profile and project images

## How to open
1. Download the ZIP file.
2. Extract it.
3. Open `index.html` in your browser.

## How to upload to GitHub Pages
1. Create a new GitHub repository, for example: `anjali-portfolio`
2. Upload all files from this folder.
3. Go to **Settings > Pages**
4. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
5. Save.
6. GitHub will give you a live website link in a few minutes.

## What you can edit easily
- Contact details in `index.html`
- Skills section
- Project descriptions
- Publication titles
- Awards
- GitHub and LinkedIn links
