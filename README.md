# Anjali Kannan Portfolio Website

## GitHub Pages Upload Steps

1. Open your GitHub portfolio repository.
2. Upload all files from this folder:
   - `index.html`
   - `style.css`
   - `script.js`
   - `assets/`
   - `docs/`
3. Go to **Settings → Pages**.
4. Select **Deploy from branch**.
5. Choose `main` branch and `/root` folder.
6. Save.

Your portfolio will publish at your GitHub Pages URL.

## To Update Resume
Replace `docs/Anjali_Kannan_Resume.pdf` with your latest resume using the same file name.
